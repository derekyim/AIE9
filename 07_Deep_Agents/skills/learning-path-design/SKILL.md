# Learning Path Design Skill

You are helping create a personalized learning path. Follow these steps:

## Step 1: Assess Current State
Understand the user's:
- Current skill level (beginner, intermediate, advanced)
- Related experience and knowledge
- Learning style preferences (video, reading, hands-on)
- Available time per week for learning
- Budget for courses and materials

## Step 2: Define Target Skills
Clarify the learning goals:
- Target skill or certification
- Proficiency level desired
- Timeline for completion
- How the skill will be applied

## Step 3: Gap Analysis
Identify:
- Prerequisites needed
- Core competencies to develop
- Nice-to-have extensions
- Related skills for synergy

## Step 4: Resource Curation
Research and recommend:
- Free resources (YouTube, documentation, tutorials)
- Paid courses (Coursera, Udemy, LinkedIn Learning)
- Books and reading materials
- Practice platforms and sandboxes
- Communities and forums

## Step 5: Create Learning Schedule
Design a structured plan with:
- Weekly milestones
- Daily/weekly time blocks
- Practice exercises
- Project checkpoints
- Assessment moments

## Step 6: Build Portfolio Evidence
Recommend projects that:
- Demonstrate learned skills
- Can be shared publicly
- Build towards certification
- Solve real problems

## Step 7: Save Outputs
Write the learning path to `workspace/learning_path.md`.

## Learning Best Practices
- Spaced repetition for retention
- Active recall over passive review
- Teach others to solidify understanding
- Build projects, not just consume content
- Track progress and celebrate wins
