# Skill Gap Assessment Skill

You are helping identify and prioritize skill gaps. Follow these steps:

## Step 1: Define Career Target
Understand:
- Target role or career path
- Industry and company type
- Timeline for transition
- Salary/level expectations

## Step 2: Research Target Requirements
Analyze job postings to identify:
- Required technical skills
- Required soft skills
- Preferred qualifications
- Common tools and technologies
- Certifications valued

## Step 3: Inventory Current Skills
Document user's existing:
- Technical skills (rate 1-5 proficiency)
- Soft skills (rate 1-5 proficiency)
- Tools and technologies
- Certifications and credentials
- Relevant experience

## Step 4: Gap Analysis Matrix
Create a comparison showing:
| Skill | Required Level | Current Level | Gap | Priority |
- Categorize gaps as: Critical, Important, Nice-to-have

## Step 5: Prioritization Framework
Rank skills by:
- Frequency in job postings (high demand)
- Difficulty to acquire (quick wins vs long-term)
- Transferability (value across roles)
- Personal interest (sustainability)

## Step 6: Development Recommendations
For each priority skill, suggest:
- Learning resources
- Time investment needed
- Ways to demonstrate competency
- Milestones for progress

## Step 7: Save Outputs
Write the assessment to `workspace/skill_gap_assessment.md`.

## Priority Categories
- **Immediate (0-3 months)**: Critical gaps, quick wins
- **Short-term (3-6 months)**: Important skills with moderate effort
- **Long-term (6-12 months)**: Advanced skills, certifications
