# Session 17: 🔀 MCP Servers & A2A

🎯 **Goal**: Learn to set up MCP servers and enable public communication between agents.

📚 **Learning Outcomes**
- Understand how to set up your own MCP servers 
- Understand Agent2Agent Protocol (A2A) and how to enable A2A communications between remote and client agents through agent cards and MCP

🧰 **New Tools**
- Agent2Agent: [A2A](https://github.com/a2aproject/A2A)
- ChatGPT Apps (MCP integration)
- [ngrok](https://ngrok.com/)
- OAuth

**MCP vs A2A**
